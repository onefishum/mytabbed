# mytabbed

## 加载st
`tabbed -c -r 2 st -w ''`
  
## 设置默认terminal为st
`gsettings set org.gnome.desktop.default-applications.terminal exec "/usr/local/bin/tabbed -c -r 2 /usr/local/bin/st -w ''"`
